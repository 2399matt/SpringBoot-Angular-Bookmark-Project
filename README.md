
# Bookmark Project

Bookmark project created with SpringBoot + Angular. Users can submit a URL and appropriate tags, and have a bookmark created and displayed for them. Bookmarks contain a url, title, description, timestamp, favicon, and view tracker. This project was used to get more familiar with Angular. (Disclaimer: Tailwind css generated by AI).

#


## Installation

To deploy this project, configure the .env file, and angular environment files (examples provided). Then run: 

```bash
  docker compose up --build
```


## Libraries Used
- [Jsoup](https://jsoup.org/) – HTML parser for Java. 
- [Spring Boot](https://spring.io/projects/spring-boot) – Java backend framework. 
- [Angular](https://angular.io/) – Frontend framework. 
- [Auth0 Angular SDK](https://auth0.com/docs/quickstart/spa/angular) – Authentication library. 

## Tools / Infrastructure
- [Nginx](https://www.nginx.com/) – Reverse proxy / web server. 
- [Docker](https://www.docker.com/) – Containerization platform. 
- [MySQL](https://www.mysql.com/) – Database. 
- [Auth0](https://Auth0.com) – Authorization Server.


## Features

- Submit and display bookmarks with metadata.
- Tag bookmarks for easier categorization.
- Search for bookmarks.
- Inline edit of bookmarks (description, title).
- OAuth authentication using Auth0 and Spring Boot as an OAuth Resource Server.
- Nginx used to serve Angular application and act as a reverse proxy for backend API calls.
